OpenBazaar Contract Tools

This is how it works:

OpenBazaar Ricardian Contracts are formatted in JSON, but this can be difficult for ordinary folk to read. This tool imports the raw contract and transforms it into an easy-to-understand format. Paste in your raw JSON-formatted contract below and press 'Import'.

This tool isn't compatible with contracts used in the current client. Rather, this tool is to test out the new schema we've designed for the OpenBazaar Ricardian Contracts. You can press the 'Example' button to see what this looks like in it's raw form before importing it.
